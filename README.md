1. 프로젝트 개요

성격: 대외활동 실습 및 팀/개인 과제 기록

주요 언어: Java 17

빌드 도구: Maven

학습 목표: Java를 활용한 비즈니스 로직 구현 및 백엔드 개발 프로세스 숙달

2. 주요 학습 및 구현 내용

객체지향 5원칙(SOLID)의 이해와 적용: 유지보수가 쉬운 코드 작성 실습

Java Collection Framework 활용: List, Map, Set 등을 이용한 데이터 관리 로직 구현

예외 처리(Exception Handling): 견고한 애플리케이션 개발을 위한 예외 처리 전략 학습

비즈니스 요구사항 분석: 실제 서비스에서 쓰이는 회원 관리, 게시판 등의 기본 로직 설계

3. 기술 스택 (Tech Stack)

Language: Java

Build Tool: Maven

Environment: IntelliJ IDEA, Git, GitHub

4. 프로젝트 구조

src/main/java: 학습한 주요 기능 및 알고리즘 구현 소스 코드

src/test/java: 단위 테스트(Unit Test)를 통한 코드 검증 기록

pom.xml: Maven 기반의 의존성 관리 및 빌드 설정

5. 성장 포인트

단순히 문법을 익히는 것에 그치지 않고, Maven을 이용한 프로젝트 관리 체계와 협업 환경에서의 Git 사용법을 익혔습니다.

코드 리뷰와 실습을 통해 더 효율적이고 가독성 높은 코드를 작성하는 능력을 배양했습니다.
